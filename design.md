# Paris Tourism Website - Design Style Guide

## Design Philosophy

### Visual Language
- **Elegant Parisian Aesthetic**: Sophisticated and romantic design inspired by Parisian chic
- **Artistic Heritage**: Drawing from French artistic traditions - Impressionism, Art Nouveau, and Haussmannian architecture
- **Timeless Elegance**: Classic design elements that evoke the eternal charm of Paris
- **Cultural Authenticity**: Authentic French visual elements without clichés

### Color Palette
- **Primary Colors**:
  - Parisian Blue: #4A90E2 (inspired by French blue and Yves Klein blue)
  - Cream White: #F8F6F0 (warm white like French vanilla)
  - Charcoal Gray: #2C3E50 (sophisticated dark tone)
- **Accent Colors**:
  - Rose Gold: #E8B4B8 (subtle pink reminiscent of Parisian sunsets)
  - Sage Green: #9CAF88 (inspired by Parisian gardens)
  - Warm Gold: #D4AF37 (elegant gold accents)

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings, inspired by French typography
- **Body Font**: "Inter" - Clean, modern sans-serif for readability
- **Accent Font**: "Dancing Script" - For French text elements and decorative purposes

## Visual Effects & Styling

### Hero Section Design
- **Background**: Cinematic hero image of Paris skyline at golden hour
- **Text Animation**: Typewriter effect for main heading using Typed.js
- **Overlay**: Subtle gradient overlay (40% opacity) for text readability
- **Parallax**: Gentle parallax scrolling effect on hero image

### Interactive Elements
- **Hover Effects**: 
  - 3D tilt effect on attraction cards using CSS transforms
  - Color morphing on buttons (Parisian Blue to Rose Gold)
  - Subtle glow effects on interactive elements
- **Scroll Animations**: 
  - Fade-in with 20px upward translation
  - Staggered animations for card grids
  - Smooth transitions using Anime.js

### Visual Effects Implementation
- **Background**: Subtle aurora gradient flow using CSS animations
- **Text Effects**: 
  - Gradient text animation on key headings
  - Character shuffle effect for dynamic content
  - Outline-to-fill text animation
- **Image Effects**:
  - Ken Burns pan/zoom on hero images
  - Infinite image scroller for attraction galleries
  - Smooth transitions with Splide carousel

### Layout & Structure
- **Grid System**: CSS Grid with Tailwind utilities
- **Spacing**: Generous whitespace following French design principles
- **Breakpoints**: Mobile-first responsive design
- **Navigation**: Fixed header with elegant transparency effects

### Cultural Design Elements
- **French Patterns**: Subtle fleur-de-lis motifs as decorative elements
- **Architectural Inspiration**: Haussmannian building facades as section backgrounds
- **Artistic References**: Impressionist color palettes in gradient effects
- **Parisian Icons**: Stylized representations of landmarks (Eiffel Tower, Arc de Triomphe)

### Interactive Components Styling
- **Chatbot**: Floating design with Parisian Blue accent and smooth animations
- **Language Switcher**: Elegant toggle with flag icons and smooth transitions
- **Image Galleries**: Masonry layout with hover zoom effects
- **Forms**: Clean, minimal design with French-inspired input styling

### Animation & Motion
- **Library Usage**:
  - Anime.js for smooth micro-interactions
  - Typed.js for typewriter effects
  - Splide for image carousels
  - ECharts for data visualization with Parisian color scheme
- **Timing**: 300ms duration for most animations (French elegance - never rushed)
- **Easing**: Custom cubic-bezier curves for natural motion
- **Performance**: Hardware-accelerated transforms for smooth 60fps animations

### Accessibility & Usability
- **Contrast**: Minimum 4.5:1 ratio for all text
- **Focus States**: Elegant focus rings with Parisian Blue
- **Dark Mode**: Sophisticated dark theme with warm undertones
- **Font Scaling**: Responsive typography with fluid scaling

### Cultural Authenticity
- **French Language Integration**: Seamless bilingual design
- **Local Aesthetics**: Avoiding tourist clichés, focusing on authentic Parisian style
- **Seasonal Adaptation**: Design elements that reflect Paris through seasons
- **Artistic Heritage**: Visual references to French art movements

This design approach creates a sophisticated, culturally authentic experience that captures the essence of Paris while providing modern functionality and interactivity.
